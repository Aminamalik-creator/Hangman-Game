# 🎯 Hangman Game

A classic **Hangman Game** built with simple and interactive logic where players guess letters to reveal a hidden word before running out of attempts.

---

## 🕹️ Gameplay

- The game randomly selects a hidden word  
- Player guesses one letter at a time  
- Correct guesses reveal the letter’s position  
- Incorrect guesses reduce remaining attempts  

### 🎮 Game Ends When:
- 🎉 Player guesses the word correctly (Win)  
- 💀 Player runs out of attempts (Lose)  

---

## ✨ Features

- 🔤 Random word selection  
- 🎮 Interactive gameplay  
- 📉 Limited attempts system  
- 🧠 Tracks guessed letters  
- 🏆 Win/Loss display  
- 💡 Beginner-friendly logic  

---

## 🛠️ Technologies Used

- Language: C  
- Concepts: Loops, Conditions, Strings
